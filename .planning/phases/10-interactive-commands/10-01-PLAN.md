  <action>
创建 `src/claw_cron/prompt.py`，包含以下函数：

1. `prompt_text(message: str, default: str | None = None) -> str`
   - 使用 `inquirer.text()` 实现文本输入
   - 支持默认值

2. `prompt_confirm(message: str, default: bool = True) -> bool`
   - 使用 `inquirer.confirm()` 实现确认提示
   - 返回布尔值

3. `prompt_select(message: str, choices: list[str], default: str | None = None) -> str`
   - 使用 `inquirer.select()` 实现单选
   - 支持默认选中

4. `prompt_multiselect(message: str, choices: list[str]) -> list[str]`
   - 使用 `inquirer.checkbox()` 实现多选
   - 返回选中项列表

5. `prompt_cron() -> str`
   - 提供常用 cron 预设选择
   - 预设列表（每个预设显示描述和表达式）：
     - "每分钟" → `* * * * *`
     - "每小时整点" → `0 * * * *`
     - "每天早上8点" → `0 8 * * *`
     - "每天中午12点" → `0 12 * * *`
     - "每天晚上6点" → `0 18 * * *`
     - "每周一早上9点" → `0 9 * * 1`
     - "工作日早上9点" → `0 9 * * 1-5`
     - "每月1号" → `0 0 1 * *`
     - "自定义" → 调用 prompt_text 输入
   - 返回 cron 表达式字符串

使用 InquirerPy 的替代语法（typed style）：
```python
from InquirerPy import inquirer
from InquirerPy.base.control import Choice

result = inquirer.text(message="Enter name:").execute()
```

注意：InquirerPy 与 Rich 的颜色配置可能冲突，使用默认样式即可。
