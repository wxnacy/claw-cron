<read_first>
- `src/claw_cron/storage.py` — 当前 Task dataclass 定义，了解现有字段顺序和类型

<action>

@dataclass
class Task:
    name: str

### Task 2: 新增 update_task 函数

<read_first>
- `src/claw_cron/storage.py` — 了解 load_tasks / save_tasks / delete_task 的实现模式
</read_first>

<action>
在 `delete_task` 函数之后新增 `update_task` 函数：

```python
def update_task(name: str, path: Path = TASKS_FILE, **kwargs: Any) -> bool:
    """Update fields of an existing task by name.

    Args:
        name: Name of the task to update.
        path: Path to the tasks YAML file.
        **kwargs: Fields to update (e.g., enabled=True, client_cmd="...").

    Returns:
        True if task was found and updated, False if not found.
    """
    tasks = load_tasks(path)
    for task in tasks:
        if task.name == name:
            for key, value in kwargs.items():
                if hasattr(task, key):
                    setattr(task, key, value)
            save_tasks(tasks, path)
            return True
    return False
```
</action>

<acceptance_criteria>
- `grep "def update_task" src/claw_cron/storage.py` 有输出
- `grep "setattr(task, key, value)" src/claw_cron/storage.py` 有输出
- `grep "return False" src/claw_cron/storage.py` 有输出（update_task 中）
- `uv run python -c "from claw_cron.storage import update_task; print('ok')"` 输出 `ok`
</acceptance_criteria>
