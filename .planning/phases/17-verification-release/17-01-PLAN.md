"""Unit tests for channel boundary conditions."""

from __future__ import annotations
import time

**TestSupportsCapture** — 验证 supports_capture 属性：
- `test_imessage_does_not_support_capture()`: `CHANNEL_REGISTRY["imessage"]().supports_capture is False`
- `test_email_does_not_support_capture()`: `CHANNEL_REGISTRY["email"]().supports_capture is False`
- `test_qqbot_supports_capture()`: `CHANNEL_REGISTRY["qqbot"]().supports_capture is True`
- `test_feishu_supports_capture()`: `CHANNEL_REGISTRY["feishu"]().supports_capture is True`
- `test_wecom_supports_capture()`: `WeComChannel().supports_capture is True`

**TestCaptureUnsupportedChannel** — 验证不支持 capture 的通道抛出 NotImplementedError：
- `test_imessage_capture_raises()`: `await CHANNEL_REGISTRY["imessage"]().capture_openid()` 抛 `NotImplementedError`，message 包含 "imessage"
- `test_email_capture_raises()`: `await CHANNEL_REGISTRY["email"]().capture_openid()` 抛 `NotImplementedError`，message 包含 "email"

**TestTokenInfo** — 验证 TokenInfo 过期逻辑（无需 mock）：
- `test_token_not_expired_when_fresh()`: `TokenInfo(access_token="t", expires_at=time.time()+3600).is_expired() is False`
- `test_token_expired_when_past_expiry()`: `TokenInfo(access_token="t", expires_at=time.time()-1).is_expired() is True`
- `test_token_expired_within_buffer()`: `TokenInfo(access_token="t", expires_at=time.time()+30, buffer_seconds=60).is_expired() is True`（30s < 60s buffer）

**TestWeComConfigValidation** — 验证配置缺失时抛出 ChannelConfigError：
- `test_missing_corp_id_raises()`: `WeComChannel(WeComConfig(agent_id=1, secret="s"))` 调用 `await ch._get_access_token()` 抛 `ChannelConfigError`
- `test_missing_secret_raises()`: `WeComChannel(WeComConfig(corp_id="c", agent_id=1))` 调用 `await ch._get_access_token()` 抛 `ChannelConfigError`

**TestWeComTokenCaching** — 验证 token 缓存和刷新：
- `test_token_cached_when_valid()`: mock `_http_client.get` 返回 `{"errcode":0,"access_token":"tok1","expires_in":7200}`，连续调用 `_get_access_token()` 两次，assert `_http_client.get` 只被调用 1 次，两次返回值均为 `"tok1"`
- `test_token_refreshed_when_expired()`: 先设置 `ch._token = TokenInfo(access_token="old", expires_at=time.time()-1)`，mock `_http_client.get` 返回 `{"errcode":0,"access_token":"new","expires_in":7200}`，调用 `_get_access_token()`，assert 返回 `"new"`

**TestWeComSendText** — 验证消息发送：
- `test_send_text_success()`: mock `_get_access_token` 返回 `"token"`，mock `_http_client.post` 返回 `{"errcode":0,"errmsg":"ok"}`，调用 `await ch.send_text("userid123", "hello")`，assert `result.success is True`，assert post payload 包含 `"touser": "userid123"` 和 `"msgtype": "text"`
- `test_send_text_strips_prefix()`: 调用 `await ch.send_text("c2c:userid123", "hello")`，assert post payload 中 `"touser": "userid123"`（去掉 `c2c:` 前缀）
- `test_send_text_rate_limit_retries()`: mock `_http_client.post` 前两次返回 `{"errcode":45009,"errmsg":"rate limit"}`，第三次返回 `{"errcode":0}`，assert `result.success is True`（tenacity retry 生效）

**TestWeComSendMarkdown** — 验证 markdown 发送：
- `test_send_markdown_success()`: mock post 返回 `{"errcode":0}`，assert `result.success is True`，assert payload `"msgtype": "markdown"`
- `test_send_markdown_fallback_to_text()`: mock post 返回 `{"errcode":50001,"errmsg":"not support markdown"}`，assert 回退调用 `send_text`（通过检查第二次 post 的 payload `"msgtype": "text"`）
