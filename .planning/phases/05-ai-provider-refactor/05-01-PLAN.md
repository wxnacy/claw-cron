# Phase 5 Plan 01: Provider Infrastructure & Configuration

## Objective

**Purpose:** Establish the foundation for multi-provider AI support with configuration management and tool abstraction.

**Output:**
- `BaseProvider` abstract class with `chat_with_tools()` interface
- `AIConfig` configuration class with environment variable support
- `ToolDefinition` and `ToolCall` dataclasses for provider-agnostic tool handling
- Exception hierarchy for provider errors

## Context

### Goal

Refactor `agent.py` from single-provider (Anthropic only) to Provider Pattern supporting multiple AI providers (Anthropic, OpenAI) with unified Tool Use support.

### Requirements Addressed

| ID | Description | Plan Coverage |
|----|-------------|---------------|
| **PROV-02** | Implement `BaseProvider` abstract base class with `chat_with_tools()` | Task 1 |
| **PROV-05** | Add `AIConfig` configuration class | Task 2 |
| **PROV-06** | Support environment variables (`CLAW_CRON_API_KEY`, `CLAW_CRON_MODEL`, `CLAW_CRON_PROVIDER`) | Task 2 |
| **TOOL-01** | Define neutral format `ToolDefinition` dataclass | Task 3 |
| **TOOL-02** | Define `ToolCall` dataclass | Task 3 |
| **TOOL-03** | Implement Tool format converters | Task 3 |

### Research Summary

From `.planning/research/ai-providers.md`:

**Key Findings:**
1. Current implementation tightly coupled to Anthropic SDK
2. Tool Use is provider-specific — Anthropic uses `input_schema`, OpenAI uses `parameters`
3. Provider pattern from agent-summary is well-designed but lacks Tool Use support
4. Configuration needs extension for provider-specific settings

**Recommended Approach:**
- Use Option B: Define tools once in neutral format, providers convert internally
- Add `ToolConverter` helper for format translation
- Use pydantic-settings for configuration management

### Current Architecture

```
src/claw_cron/
├── agent.py          # Anthropic-only, hardcoded model
├── config.py         # Simple YAML loader, no AI config
```

### Task 3: Implement Tool Abstraction & BaseProvider

**Files:**
- `src/claw_cron/providers/tools.py`
- `src/claw_cron/providers/base.py`
- `src/claw_cron/providers/__init__.py` (update exports)

**Action:**

1. **Create `src/claw_cron/providers/tools.py`:**

   ```python
   from dataclasses import dataclass
   from typing import Any

   @dataclass
   class ToolDefinition:
       """Provider-agnostic tool definition."""
       name: str
       description: str
       parameters: dict[str, Any]  # JSON Schema format

   @dataclass
   class ToolCall:
       """Provider-agnostic tool call result."""
       id: str  # Tool call ID (for response correlation)
       name: str
       arguments: dict[str, Any]

   def to_anthropic_tool(tool: ToolDefinition) -> dict:
       """Convert to Anthropic ToolParam format."""
       return {
           "name": tool.name,
           "description": tool.description,
           "input_schema": tool.parameters,  # Key rename
       }

   def to_openai_tool(tool: ToolDefinition) -> dict:
       """Convert to OpenAI function format."""
       return {
           "type": "function",
           "function": {
               "name": tool.name,
               "description": tool.description,
               "parameters": tool.parameters,
           }
       }
   ```

2. **Create `src/claw_cron/providers/base.py`:**

   ```python
   from abc import ABC, abstractmethod
   from dataclasses import dataclass
   from typing import Any

   from .tools import ToolDefinition, ToolCall

   @dataclass
   class ProviderResult:
       """Result from a provider chat call."""
       content: str  # Text content (empty if tool_use)
       tool_calls: list[ToolCall]  # Tool calls (empty if text response)
       stop_reason: str  # "end_turn" | "tool_use" | "max_tokens"
       raw_response: Any  # Provider-specific raw response (for debugging)

   class BaseProvider(ABC):
       """Abstract base class for AI providers."""

       def __init__(
           self,
           api_key: str,
           model: str,
           base_url: str | None = None,
       ) -> None:
           self.api_key = api_key
           self.model = model
           self.base_url = base_url

       @abstractmethod
       def chat_with_tools(
           self,
           messages: list[dict],
           system_prompt: str,
           tools: list[ToolDefinition],
           max_tokens: int = 1024,
       ) -> ProviderResult:
           """Send a chat request with tool support.

           Args:
               messages: Conversation history in [{"role": "user/assistant", "content": str}] format
               system_prompt: System instructions
               tools: Available tools for the model to call
               max_tokens: Maximum tokens in response

           Returns:
               ProviderResult with content, tool_calls, and stop_reason
           """
           ...

       def get_provider_name(self) -> str:
           """Return provider name for logging/debugging."""
           return self.__class__.__name__.replace("Provider", "").lower()
   ```

3. **Update `src/claw_cron/providers/__init__.py`:**

   ```python

   from .base import BaseProvider, ProviderResult
   from .tools import ToolDefinition, ToolCall, to_anthropic_tool, to_openai_tool

       "ProviderAuthError",

   ```
