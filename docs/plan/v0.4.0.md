
### 4. openim 子进程管理（独立模块）
**新增文件**：`src/claw_cron/openim_manager.py`

提取 openim 子进程生命周期管理为独立模块，供 server 和通道层共用：
- `is_running(port)` — 检查端口是否可连接
- `start(port, config_path, openim_bin)` — 启动子进程、等待就绪、写 PID 文件
- `stop(port)` — 读 PID → SIGTERM → 等待 → SIGKILL → 删 PID
- `ensure_running(port)` — 已在运行则复用，未运行则启动，返回是否由本次启动

**server 模式**：启动/停止 openim（`cmd/server.py` 集成）
**非 server 模式**（如 `run` 命令触发通知）：
- `OpenIMBaseChannel.send_text()` 中调用 `ensure_running()`
- 若由本次启动，发送完成后调用 `stop()`
- 若原本就在运行（如 server 模式），不停止

### 5. 新增通用 openim 通道基类
**文件**：`src/claw_cron/channels/openim.py`

- `channel_id` 由构造参数 `platform` 决定
