# claw-cron

AI-powered cron task manager. Describe tasks in natural language, let AI configure and run them on schedule.

## Installation

```bash
uv sync
```

## Usage

```bash
claw-cron -h          # 查看帮助
claw-cron -v          # 查看版本
claw-cron --version   # 查看版本
```

## Development

```bash
uv venv
uv sync
uv run claw-cron -h
```
